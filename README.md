# Sales Data Analysis Project 📊

This project focuses on a detailed analysis of sales data, utilizing **Exploratory Data Analysis (EDA)**, **Power BI visualizations**, and structured reporting to uncover key business insights. 

## 🎯 Objective
Analyze sales data to discover trends, optimize sales strategies, and provide actionable recommendations by investigating factors influencing sales performance across different products, regions, and customer demographics.

## 📂 Repository Contents
- **`Sales Data.xlsx`**: Raw transactions, products, customers, and regions data.
- **`EDA.xlsx`**: Comprehensive exploratory data analysis highlighting trends and patterns.
- **`Sales Analysis.pbix`**: Interactive Power BI dashboard featuring dynamic visualizations of revenue growth, regional performance, and product category trends.
- **`ER Diagram.png`**: Entity-Relationship diagram showcasing the data model structure.

## 💡 Key Insights
- Identified peak sales periods and seasonal variations.
- Mapped regional revenue contributions and high-growth markets.
- Analyzed customer purchasing behaviors to recommend targeted marketing strategies.
