# Sales-Data-Analysis

> An end-to-end sales analytics capstone: exploratory analysis in Excel plus an interactive Power BI dashboard.

A data-analysis project that investigates sales performance across products, regions, and customers to surface trends and actionable recommendations.

## Repository Contents
- **`Sales data.xlsx`** — raw transactions, product, customer, and region data
- **`EDA.xlsx`** — exploratory data analysis highlighting trends and patterns
- **`Sales Analysis.pbix`** — interactive Power BI dashboard (revenue growth, regional and category performance)
- **`ER Diagram.png`** — the underlying data model

## Key Insights
- Identified peak sales periods and seasonal variation
- Mapped regional revenue contribution and high-growth markets
- Analysed customer purchasing behaviour to inform targeted strategy

## Tools
- Microsoft Excel (EDA), Power BI (dashboarding and data modelling)
